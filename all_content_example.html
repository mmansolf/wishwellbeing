Welcome to My Website

This is a demonstration of a modular website built with HTML components using a custom includes system.

Our modular component system makes it easy to build consistent websites with reusable parts. Here are some of the benefits:

Modular Design

Components can be reused across multiple pages, ensuring consistency throughout your site.

Customizable

Each component instance can be customized with data attributes, making them flexible for different contexts.

To get started with this template, follow these steps:

  1. Download the template - Clone the repository or download the files.
  2. Customize your components - Modify the HTML files in the components directory.
  3. Add content to your pages - Create new HTML pages and include your components.
  4. Deploy to Netlify - Upload your files to Netlify for fast, reliable hosting.

For more detailed instructions, check out the documentation.
